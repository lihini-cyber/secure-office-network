# Secure Office Network – Cisco Packet Tracer

## Overview
This project simulates a small office network using VLAN segmentation and inter-VLAN routing in Cisco Packet Tracer.

## Features
- VLAN configuration
- Subnetting
- Trunk configuration
- Inter-VLAN routing
- Static IP addressing
- Connectivity testing using ping

## VLAN Structure

| Department | VLAN | Network |
|---|---|---|
| HR | 10 | 192.168.10.0/24 |
| IT | 20 | 192.168.20.0/24 |
| SALES | 30 | 192.168.30.0/24 |

## Technologies Used
- Cisco Packet Tracer
- VLANs
- IP Addressing
- Subnetting
- Router-on-a-stick routing

## Skills Demonstrated
- Network segmentation
- Router and switch configuration
- Troubleshooting
- CLI configuration
